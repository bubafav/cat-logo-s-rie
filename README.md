
O **Séries+** é um projeto de front-end criado com o objetivo de praticar e demonstrar conhecimentos em **HTML5 e CSS3**, focando em layout moderno, animações e organização visual inspirada em plataformas de streaming.

O site conta com um **banner animado**, um **carrossel infinito de séries**, filtros por categorias e seções bem definidas como Destaques, Sobre e Contato. Todo o projeto foi desenvolvido sem uso de JavaScript, explorando ao máximo os recursos do CSS, como animações, flexbox e responsividade.

 Funcionalidades:

* Layout responsivo para desktop e mobile
* Banner com efeito de movimento e overlay escuro
* Carrossel automático infinito com pausa ao passar o mouse
* Cards de séries com imagens e títulos
* Navegação simples e intuitiva

 Tecnologias utilizadas:

* HTML5
* CSS3

- Projeto desenvolvido para fins educacionais e portfólio.
